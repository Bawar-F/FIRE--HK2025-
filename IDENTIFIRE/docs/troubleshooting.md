# Troubleshooting & Solutions

When things go wrong, document the problem and solution here so we can learn.

## Common problems

### Git/GitHub issues
Problem: Can't push changes  
Solution: Make sure you're on a branch (not main) and try pulling first.

Problem: Merge conflicts  
Solution: Ask for help. Don't guess.
